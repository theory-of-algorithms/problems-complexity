\question
Determine the number of comparisons made by Bubble sort on the following inputs.
\begin{parts}
  \part $[3,2,1]$
  \part $[4,3,2,1]$
  \part $[5,4,3,2,1]$
  \part $[6,5,4,3,2,1]$
  \part $[20,19,18,\ldots,3,2,1]$
  \part $[3,4,5,2,1]$
  \part $[4,5,1,2,3]$
\end{parts}
\begin{solution}
  \begin{parts}
    \part $2 + 1 = 3$
    \part $3 + 2 + 1 = 6$
    \part $4 + 3 + 2 + 1 = 10$
    \part $5 + 4 + 3 + 2 + 1 = 15$
    \part $6 + 4 + 3 + 2 + 1 = 21$
    \part $(20 \times 19) \div 2 = 190$
    \part $4 + 3 + 2 + 1 = 10$
    \part $5 + 4 + 3 = 12$, using the fact that if no swaps are made in one pass then the list is sorted.
  \end{parts}
\end{solution}
